# gitadot

Wrapper of `git add .`

## Installation

```
npm install -g nimstall
nimstall -g gitadot
```

## Usage

### Command Line

```
gitadot
```

## Common Workflow

First install..

```
nimstall -g gitcup
```

Then do..

```
mkdir my-project
cd my-project
git init
touch Readme.md
gitadot
gitcup
git push
```
